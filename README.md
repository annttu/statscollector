statscollector
==============

Työkalu networktesterin lähettämien statistiikkojen keräämiseen.
Tallentaa saadut statsit tietokantaan.

Asennus
=======

    virtualenv env --python=python3.4
    . env/bin/activate
    pip install -r requirements.txt    

Käyttö
=======

    python statscollector/app.py
